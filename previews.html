Venetian Blinds Card – Layout Previews

Three layouts ship in the card and are selectable in the visual editor via layout: cards | list | segmented. All show tilt-only control with configurable preset buttons (no slider). Default 4 presets, max 6. The card uses CSS container queries so labels collapse to icons-only when the card is narrower than responsive_breakpoint (default 320px) — see the bottom row.
